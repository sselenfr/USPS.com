## Design Standards

The HTML prototype follows:

- **[USPS Digital Style Guide](https://www.usps.com/styleguide/)** — colors, typography, spacing, buttons, and navigation
- **[U.S. Web Design System (USWDS)](https://designsystem.digital.gov/)** — federal UX and accessibility best practices
- **Section 508 / WCAG 2.1 AA** — accessible markup, focus management, ARIA patterns

### Key Brand Tokens (usps.com/styleguide)

| Token | Value | Usage |
|---|---|---|
| USPS Digital Blue | `#333366` | Primary brand color, headings, nav, buttons |
| White | `#FFFFFF` | Backgrounds |
| USPS Digital Red | `#E71921` | Alerts, error messaging, limited accents |
| USPS Digital Green | `#218748` | Ecommerce / transaction buttons |
| USPS Accent Blue | `#3573B1` | Inline links, graphics |
| Light BG Gray | `#F7F7F7` | Section backgrounds |
| Text Gray | `#595959` | Secondary text |
| Primary Font | Arial / Helvetica | All digital copy |
